I had a need to write some code recently, and ran into the "Rails is a pain to set up" issue that people have been talking about recently. It's gotten bad enough that Yehuda Katz, a member of the Ruby on Rails core team, has set up a Kickstarter project, "Rails.app", to address the issue.

Since I'm more familiar with running services on something more like an Amazon EC2 instance, I looked around for something I could put some time and secret sauce into that would give me ease of use and reusability.

That's when I found Vagrant.

Vagrant is a way to spin up VMs with standardized environments very quickly. It depends on the wonderful Oracle Virtualbox, and can even integrate Chef or Puppet.

I'll write a bit more about my experiance with Vagrant soon. You should start looking at it now, though.
